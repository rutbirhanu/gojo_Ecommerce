# gojo_Ecommerce


## Prerequisites
- Node.js 
- npm 
- MongoDB
- Firebase account


## Installation
  - cd Backend ====> npm install
  - cd Gojo_Frontend  =====> npm install


## Set up environment variables
  - MONGODB_CONNECTION=YOUR_MONGODB_CONNECTION_STRING
  - JWT_SECRET=YOUR_JWT_SECRET
  - VITE_API_MAPBOX_TOKEN (on the frontend)
